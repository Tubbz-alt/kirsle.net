Kirsle.net Solar (Bootstrap)

This is another design refresh for Kirsle.net Solar that uses Twitter Bootstrap as its layout engine, making it work better on mobile. Resize the browser window and see how the site reacts.

This design went live on Kirsle.net on October 15, 2014. It replaced the previous version of the Solar theme.
